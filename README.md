# sbgmodloader.github.io

Official landing page for the Super Battle Golf Mod Loader ecosystem.

**Live Site:** https://sbgmodloader.com

## Custom Domain Setup

This repository uses a custom domain (sbgmodloader.com) configured via:
- GitHub Pages (Settings → Pages)
- CNAME file pointing to sbgmodloader.com
- DNS CNAME record at domain registrar

## Local Development

Simply open `index.html` in a web browser.

## Deployment

Changes pushed to `main` branch are automatically deployed to GitHub Pages.
